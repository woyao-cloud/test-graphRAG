# RAG 实战指南 — 从原理到企业级落地

## 关于本书

本书是一本系统化的 RAG（Retrieval-Augmented Generation）技术实战指南，从基础原理到企业级落地，覆盖了 RAG 系统的完整生命周期。

## 目标读者

- **AI 应用开发者**：希望将 RAG 集成到产品中的工程师
- **算法工程师**：希望优化 RAG 管道的检索与生成效果
- **技术架构师**：需要设计企业级知识库系统的架构师
- **技术管理者**：需要评估 RAG 技术选型和团队协作的管理者
