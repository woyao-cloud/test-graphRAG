# React 19 精通实战 — 从原理到企业级应用

## 关于本书

本书是一本系统化的 React 19 实操指南，从 Fiber 工作机制到 Hooks 原理，从性能优化到企业级应用架构，覆盖 React 开发的完整知识体系。

## 章节概览

| 章节 | 主题 | 核心内容 |
|------|------|---------|
| Ch1 | React 19 概述与生态 | 新特性、框架对比、迁移指南 |
| Ch2 | Fiber 工作机制 | 协调算法、双缓冲、优先级调度 |
| Ch3 | Hook 原理与分类 | useState、useEffect、useRef 等原理 |
| Ch4 | Hook 深入与实践 | 自定义 Hook、组合模式、测试 |
| Ch5 | 性能优化 | memo、虚拟化、代码分割、Profiler |
| Ch6 | 典型问题处理模式 | 防抖节流、错误边界、Portal |
| Ch7 | 开发者必备技能 | TypeScript、测试、调试、安全 |
| Ch8 | 生态与工具链 | 状态管理、路由、样式方案 |
| Ch9 | 企业级应用架构 | 微前端、设计系统、CI/CD |
| Ch10 | 高级进阶与实战 | Actions、Server Components、并发模式 |

## 配套代码
