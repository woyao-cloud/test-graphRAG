2. 评估方法

方法一：实体命中率（自动）

对每个问题，检查答案中是否包含预期实体：

def evaluate_query(question, expected_entities, response):
    """检查答案是否包含预期实体"""
    found = [e for e in expected_entities if e in response.answer]
    recall = len(found) / len(expected_entities)
    return recall

方法二：关键词匹配（自动）

检查答案是否包含预期关键词：

def evaluate_keywords(question, expected_keywords, response):
    found = [k for k in expected_keywords if k in response.answer]
    return len(found) / len(expected_keywords)
